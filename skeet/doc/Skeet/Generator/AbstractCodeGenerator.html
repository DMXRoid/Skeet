A list of generated code templates, and their corresponding  settings.  In the format:


array( 		"template_type" => AbstractCodeGenerator::GENERATED_CODE_TEMPLATE_TYPE_(INDIVIDUAL|ALL) 		"component_name" => "", // the name the component, passed to ComponentFactory::getComponent()         "path" => "", // the path below Skeet::getConfig("application_lib_path") to store the generated code in         "file_suffix" => "", // the suffix for files written.         "overwrite_existing_file => true|false, // whether to overwrite an existing file on generation of code         "settings" => array() // an array of arbitrary settings to pass to the template component  )
